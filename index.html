So what the hecc is Puppertino?

Puppertino is a CSS framework, that mimics the look of MacOS and Human Guidelinnes, while adding it's own taste on things.
The focus of Puppertino is help you create a beautiful looking App or Website, and being small sized, cool looking and modular.

Puppertino allows you to use it with Bootstrap, SkeletonCSS, Flexbox Grid or any other major starting point for websites, without getting in the way. This also means that you don't have to expect Puppertino to have grids, rows or any of that.

Oh ok, so is just a MacOS theme?

Nope, I want to think of it like a theme for any starting point framework that allows you to create something that looks great without having the pain of it affecting the performance of your website. This means that there will be components that will be added by hand and not because they are from MacOS, in the same way, not every thing in the MacOS interface will be added to Puppertino. If I think people won't use it, I will not add it.

Focused on speed and performance

Yeah, yeah, I've said this already, but the thing is: Pupperino is in development and it will be on it for a long time, and the main goal is to make it as lightweight as possible, so any other framework like Animate.css won't be included in Pupperino. And you should be able to choose if you want only a certain part of Pupperino or if you want all the components.

Pupperino currently only needs the Raleway font from Google Fonts, but you can change it by changing the --font variable in the CSS.

Pupperino (with all of it's components) should be able to fit on a Floppy disk, how retro!
